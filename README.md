# Gloom for Zed &nbsp; [![GitHub stars](https://img.shields.io/github/stars/intemperie/gloom-zed.svg?style=social&label=Star)](https://github.com/intemperie/gloom-zed/stargazers)
<sup>For [Zed](https://zed.dev/)</sup>
